# Online Shopping Store Project

A Java console application that provides an online shopping system for clothes and makeup items.  
It allows users to:
- View items available in the store
- Buy clothes and makeup
- Admin users can add or delete items

## Features
- Console-based user interface
- User roles: Admin and Customer
- File-based storage for items
- Basic object-oriented structure using classes and inheritance

## How to Run
1. Compile the Java file:
   ```bash
   javac OnlineShoppingStore\OnlineShoppingStore.java
